Bem-vindo ao Meu Projeto de Estudos

Este é um projeto de aprendizado em desenvolvimento web com Flask, MongoDB, Docker e REST API.

Sobre o Projeto

Esta aplicação é parte dos meus estudos para explorar a criação de aplicações web modernas. Utilizo o framework Flask para construir uma API REST, MongoDB como banco de dados NoSQL, e Docker para orquestração de contêineres, garantindo um ambiente de desenvolvimento consistente.

A API fornece endpoints para acessar dados como metadados de proteínas, sequências, descrições funcionais e dados taxonômicos. Confira a documentação da API para mais detalhes.

Tecnologias Utilizadas

  • Flask: Framework Python para desenvolvimento web e APIs REST.
  • MongoDB: Banco de dados NoSQL para armazenamento de dados dinâmicos.
  • Docker: Contêineres para facilitar o desenvolvimento e implantação.
  • REST API: Endpoints para interação com dados via HTTP.
