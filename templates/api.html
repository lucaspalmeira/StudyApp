Endpoints disponíveis

GET /api/items/metadata

Retorna todos os metadados (proteínas).

curl http://localhost:5000/api/items/metadata

GET /api/items/metadata/<ec_number>

Filtra metadados por número EC.

curl http://localhost:5000/api/items/metadata/3.2.1.7

GET /api/items/sequences

Retorna todas as sequências.

curl http://localhost:5000/api/items/sequences

GET /api/items/sequences/<ec_number>

Retorna sequências filtradas por EC.

curl http://localhost:5000/api/items/sequences/3.2.1.7

GET /api/items/descriptors

Retorna todas as descrições funcionais.

curl http://localhost:5000/api/items/descriptors

GET /api/items/descriptors/<ec_number>

Retorna descrições por EC.

curl http://localhost:5000/api/items/descriptors/3.2.1.7

GET /api/items/taxon

Retorna todos os dados taxonômicos.

curl http://localhost:5000/api/items/taxon

GET /api/items/taxon/<ec_number>

Retorna dados taxonômicos por EC.

curl http://localhost:5000/api/items/taxon/3.2.1.7

Como fazer requisições

Via terminal (curl)

curl http://localhost:5000/api/items/metadata

Via Python (usando requests)


import requests
response = requests.get("http://localhost:5000/api/items/metadata")
print(response.json())
